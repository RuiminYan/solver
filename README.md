## 块的定义

### 角块
状态值 = 3 * idx + orientation (0,1,2)

| 索引 (idx) | 物理 ID | 初始位置  | 初始状态值 (ori=0) |
| :--- | :--- | :--- | :--- |
| 0 | C0 | UBL | 0 |
| 1 | C1 | UBR | 3 |
| 2 | C2 | UFR | 6 |
| 3 | C3 | UFL | 9 |
| 4 | C4 | DBL | 12 |
| 5 | C5 | DBR | 15 |
| 6 | C6 | DFR | 18 |
| 7 | C7 | DFL | 21 |

其中C4,C5,C6,C7是F2L角块,C4,C5是目标槽位的F2L角块

### 棱块
状态值 = 2 * idx + orientation (0,1)

| 索引 (idx) | 物理 ID | 初始位置 | 初始状态值 (ori=0) |
| :--- | :--- | :--- | :--- |
| 0 | E0 | BL | 0 |
| 1 | E1 | BR | 2 |
| 2 | E2 | FR | 4 |
| 3 | E3 | FL | 6 |
| 4 | E4 | UB | 8 |
| 5 | E5 | UR | 10 |
| 6 | E6 | UF | 12 |
| 7 | E7 | UL | 14 |
| 8 | E8 | DB | 16 |
| 9 | E9 | DR | 18 |
| 10 | E10 | DF | 20 |
| 11 | E11 | DL | 22 |

其中E0,E1,E2,E3是F2L棱块, E0,E1是目标槽位的F2L棱块，E8,E9,E10,E11是组成cross的棱块

Slot 1: BR (C5 + E1)
Slot 2: FR (C6 + E2)
Slot 3: FL (C7 + E3)

## Pseudo Analyzer

### Pseudo Cross
底层 4 个十字棱块 (E8, E9, E10, E11) 彼此之间的相对位置和色向正确（即形成了正确的十字形状），但允许该十字结构相对于固定的中心块存在 D 层偏移（Offset）。
* **特征**：Cross 处于 Solved, D, D', 或 D2 四种状态之一。

### Pseudo XCross
Pseudo Cross + 1 个底层角块 (F2L Corner) + 1 个中层棱块 (F2L Edge)。
	* 角块选择：$C(4, 1) = 4$ 种。
    * 棱块选择：$C(4, 1) = 4$ 种。
    * 组合总数：$4 \times 4 = 16$ 种目标状态。

### Pseudo XXCross
Pseudo Cross + 2 个底层角块 + 2 个中层棱块。
	* 角块选择：$C(4, 2) = 6$ 种。
    * 棱块选择：$C(4, 2) = 6$ 种。
    * 组合总数: $6 \times 6 = 36$ 种目标状态。

### Pseudo XXXCross
Pseudo Cross + 3 个底层角块 + 3 个中层棱块。
	* 角块选择：$C(4, 3) = 4$ 种。
    * 棱块选择：$C(4, 3) = 4$ 种。
    * 组合总数：$4 \times 4 = 16$ 种目标状态。

### 非匹配性 (Non-matching Property)
上述组合计算揭示了 Pseudo 搜索的核心特性：**索引独立 (Index Independence)**。
* **定义**：选定角块的物理 ID 集合 $\{C_a, ...\}$ 与选定棱块的物理 ID 集合 $\{E_x, ...\}$ **无需一致**。
* **示例**：在 Pseudo XXCross 中，一种合法的目标状态可以是“角块 {0, 1}”搭配“棱块 {2, 3}”。系统只关心选定的 $N$ 个角块和 $N$ 个棱块是否分别归位，而不强制要求它们组成传统的“F2L Pair”。

### 位置相对性 (Positional Relativity)
所有选定的F2L块必须满足以下特定的位置约束（参考系解耦）：

1.  **角块 (Corner) $\rightarrow$ 伪位 (Pseudo-pos)**
    * **定义**：必须相对于 **Pseudo Cross** 的当前位置是复原的。
    * **规则**：如果 Pseudo Cross 整体做了 $D$ 偏移，目标角块也必须处于偏移 $D$ 后的物理位置。

2.  **棱块 (Edge) $\rightarrow$ 固定位 (Fixed-pos)**
    * **定义**：必须相对于 **固定中心 (Centers)** 是复原的。
    * **规则**：中层棱块 ($E_0-E_3$) 不随 $D$ 层偏移，必须归位到真实的物理槽位。

### 独立性 (Independence)
由上述规则推导出的搜索策略：
* 算法在生成启发式表格或进行搜索时，应当将角块的复原状态与棱块的复原状态视为两个**正交**的维度。
* 只要选定的角块集合满足“伪位归位”，且选定的棱块集合满足“固定位归位”，即视为达成 Pseudo X..XC 状态，无需考虑它们在空间上是否相邻或匹配。


## Pair Analyzer

### Pair
称由相同槽位的F2L棱块和F2L角块组成的的棱角对为 Pair , 如果：
该棱角对可通过 Setup (调整 U 层) + Insert (插入) 的转动组合，进入目标槽位并还原。

* Setup: NULL (不转), U, U2, U'
* Insert: (以 BL 槽为例): NULL, L U L', L U' L', B' U B, B' U' B
    注：对于其他槽位（BR, FR, FL）类似，使用相应的对称/镜像公式.

### Cross + Pair
* 状态描述: Cross 已还原，且剩余 4 个 F2L 槽位中存在 Pair。

### XCross + Pair
* 状态描述: Cross 加上 1 个槽位已还原，且剩余 3 个 F2L 槽位中存在 Pair。

### XXCross + Pair
* 状态描述: Cross 加上 2 个槽位已还原，且剩余 2 个 F2L 槽位中存在 Pair。

### XXXCross + Pair
* 状态描述: Cross 加上 3 个槽位已还原，且剩余 2 个 F2L 槽位中存在 Pair。


## Pseudo Pair Analyzer

### Pseudo Pair
称由任意两个非绑定的F2L棱块和F2L角块组成的的棱角对为 Pseudo Pair , 如果：
该棱角对可通过 Setup (调整 U 层) + Insert (插入) 的转动组合，棱块进入 **固定位** 并还原, 角块进入 **伪位** 并还原.

**1. 空间定义 (继承 Pseudo Analyzer)**
* **坐标系**：允许 Cross 处于 D/D'/D2 偏移状态。
* **配对规则**：完全解耦。指定的棱块 $E$ 和角块 $C$ 不需要来自同一个物理槽位（例如：可以用 FR 的棱配 BL 的角）。
* **归位标准**：棱块对齐绝对中心，角块对齐偏移后的 Cross。

**2. 目标状态 (继承 Pair Analyzer)**
不要寻找“已归位”的状态。寻找 **Pseudo Pair 就绪** 状态。即：当前状态 $\xrightarrow{Setup (U) + Insert}$ 能够立刻满足上述的 **Pseudo 归位标准**。

* Setup: NULL (不转), U, U2, U'
* Insert: (以 BL 槽为例): NULL, L U L', L U' L', B' U B, B' U' B
    注：对于其他槽位（BR, FR, FL）类似，使用相应的对称/镜像公式.

## EO Cross Analyzer

程序计算以下四个阶段的最少步

### EO + Cross
Cross 已还原，且剩余8个棱块色向正确。

### EO + XCross
XCross 已还原，且剩余7个棱块色向正确。

### EO + XXCross
XXCross 已还原，且剩余6个棱块色向正确。

### EO + XXXCross
XXXCross 已还原，且剩余5个棱块色向正确。

### EO + XXXXCross (尚未加入此功能)
XXXXCross 已还原，且剩余4个棱块色向正确。

程序通过处理 **12 个对称状态** 来模拟颜色中性（Color Neutrality）并压缩计算量：
- **轴向覆盖**: 选取 6 个面作为底面的轴向（U, D, L, R, F, B）。
- **状态归约**: 每个轴向测试 2 个特定的姿态（原始位置及 $y$ 轴旋转）.
- **输出处理**: 最终结果会对每对状态（同一个底面的 2 个视角）取 `std::min`，输出该底面轴向下的最优步数。
